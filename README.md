
## Instalación


```bash
composer install
```

## Version de PHP
```bash
 7.4.33
```
## Version de Laravel
```bash
 8.75
```

## Correr Migración

```bash
 php artisan migrate
```

## Ejecutar Seeder

```bash
 php artisan db:seed --class=UserTableSeeder
```
## Arrancar proyecto

```bash
 php artisan serve
```

## Acceso a proyecto en navegador

```bash
 http://localhost:8000/
```
